# homework_1

Create a simple login screen in Flutter

## Screen Requirements

The screen should contain:

- A logo (image from assets)
- A text field for the username
- A text field for the password
- A checkbox ("Remember me")
- A login button

## Implementation Details

Requirements:

- Use `Column`, `SizedBox`, and basic layout widgets
- Structure the layout cleanly and make it readable
- No functionality required (UI only)
